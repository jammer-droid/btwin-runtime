## Review Criteria
1. Does the code do what the task asks? (correctness)
2. Are there tests? Do they cover edge cases?
3. Is the code readable and maintainable?
4. Are there security concerns? (injection, XSS, etc.)
5. Does it follow existing codebase patterns?
6. Are there unnecessary changes or over-engineering?
